# Apollo

A delightful local music player for iOS.

![Language](https://img.shields.io/badge/language-swift-orange)
![Platform](https://img.shields.io/badge/platform-iOS-lightgray)

## Requirements

* iOS 26.0 and later

## Trivia
* The app is named after [Apollo](https://en.wikipedia.org/wiki/Apollo), the Greek god of music.
* The first three songs played using Apollo were (in order) "[EASY](https://en.wikipedia.org/wiki/Easy_(Le_Sserafim_song))", "[CRAZY](https://en.wikipedia.org/wiki/Crazy_(Le_Sserafim_song))", and "[HOT](https://en.wikipedia.org/wiki/Hot_(Le_Sserafim_song))" by [LE SSERAFIM](https://en.wikipedia.org/wiki/Le_Sserafim). This was in reference to the South Korean girl group's [EASY CRAZY HOT Tour](https://en.wikipedia.org/wiki/Easy_Crazy_Hot_Tour).
